# skills

Skill installation remains a first-class onboarding path in the migration.

## Responsibilities

- detect supported AI coding agents in the user home directory
- install bundled `SKILL.md` files into each agent-specific skills directory
- keep the `officekit` name and OfficeCLI lineage visible in installed skill content

## Supported bundles

- `officekit`
- `officekit-docx`
- `officekit-xlsx`
- `officekit-pptx`
